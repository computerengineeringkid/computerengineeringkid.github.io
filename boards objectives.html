ADVANCING COMPUTER SCIENCE DEGREE OBJECTIVES

  1. 1.Follow a software development process to analyze a problem and to design, build and test software solutions.

  2. 2.Demonstrate software development skills using more than one programming language and development environment.

  3. 3.Implement data-driven solutions.

  4. 4.Document the planning and development of software applications using industry standard tools, techniques and processes.

  5. 5.Within software solutions, describe, implement and analyze data structure techniques such as lists, trees, hash tables and graphs, along with sorting and searching algorithms.
